  • cryptoMethod(action: string, filename: string, options?: CryptoOptions): any
    • Defined in cryptohelper.ts:43

    Helper to encrypt or decrypt settings files. The default encryption key is derived from the unique machine ID, so ideally you should change to your desired secret and strong key. Same applies for the default IV. You can also set them via the SMU_CRYPTO_KEY and SMU_CRYPTO_IV environment variables. The default cipher algorithm is AES 256. Failure to encrypt or decrypt will throw an exception.

    Parameters

    • action: string

      Action can be "encrypt" or "decrypt".

    • filename: string

      The file to be encrypted or decrypted.

    • Optional options: CryptoOptions

      Encryption options with cipher, key and IV.

    Returns any

    The (de)encrypted JSON object.
